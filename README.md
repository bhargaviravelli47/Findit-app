# FindIt App

FindIt is a Lost and Found mobile application designed to help users report, search, and recover missing items easily.

## Features
- Report Lost Items
- Upload Item Images
- Found Item Listings
- Search Functionality
- Simple UI Design

## Technologies Used
- React Native
- Expo
- JavaScript
